How do I make a Order?

Choose the shop from which you want to buy products and then select the items using the sort by catagories, search products by name into the shop and put them into your cart with quntity you can change any time, then tap the "Place Order" button in the cart.

How do I share a shop?

In the home page screen where you see shop list you will be able to find the three dots on the left top corner of the shop picture click on that and you will get option to share shop. Your shareable link will be genrated.

Having trouble creating an account?

To successfully create a samriddh user account, you must use a mobile phone number. After typing in your phone number to create an account, you will be sent an authentication code via text message. If you do not receive a code, or the code has timed out, you can tap the 'Resend Code' to receive your code again.
